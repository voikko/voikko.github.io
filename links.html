Additional reading

All of the links on this page point to resources hosted elsewhere. We do not take any responsibility of the content behind them. If you find broken links on this page or would like to suggest adding new links, please let us know.

  • About Voikko
    • Kesäkoodi 2006 final report by Harri Pitkänen (Contains some information about the history of Voikko.)
  • About related subjects
    • Iso suomen kielioppi, verkkoversio by Auli Hakulinen, Maria Vilkuna, Riitta Korhonen, Vesa Koivisto and Irja Alho. A comprehensive descriptive grammar for modern Finnish.
    • Automatic non-standard hyphenation in OpenOffice.org by László Németh.
    • Leveraging the open source ispell codebase for minority language analysis by László Németh et al. (This is mostly about Hunspell.)
    • Computational Complexity of Left-Associative Grammar by Roland Hausser. (This explains the theory behind malaga.)
    • Applications of Finite-State Transducers in Natural-Language Processing by Lauri Karttunen.
  • Other projects that have created Finnish linguistic tools or data and released it as free software
    • Sukija, a text indexing system that Voikko was based on. Voikko still shares a lot of code and most of its vocabulary with Sukija.
    • Omorfi, open morphology for Finnish language based on finite state transducers (HFST).
    • FinFlect, a library for inflecting Finnish nouns. The system is rule based and therefore not that useful in applications where high level of correctness is needed. But it is small.
    • A large word list with inflection information from the Research Institute for the Languages of Finland.
    • A Finnish Myspell dictionary. This dictionary is not based on the older Ispell dictionary. Actually it is just a list of words with no inflection information.
    • eSpeak speech synthesis system ships with quite usable Finnish voices.
    • Suopuhe, Finnish voices for the Festival speech synthesis system. This project was completed with funding from Tekes and does not seem to be active anymore.
    • The Finnish Ispell dictionary, which has also been converted into Aspell and Myspell formats. The original has not been updated since September 2000, and not much has been done with the derived dictionaries either.
  • Services and freely accessible demonstrations for some of the software listed above
    • Interactive Suomi-Malaga Demo, a demonstration of morphological analysis with Sukija variant of Suomi-malaga. The results are largely the same when Voikko variant is used.
