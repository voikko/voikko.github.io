Voikko
Installation from source on Mac OS X

The following notes are information for advanced users only. They have been assembled on Mac OS X version 10.5. They are now badly outdated and overly complicated (depending on what exactly you want to achieve). Everyone building on OS X usually wants to write their own build instructions instead of updating the one below :) The bottom line is: asking for help on the mailing list before starting may save you a lot of trouble and we really need someone to update these instructions.

Things that work:

  • libvoikko and its dependecies can be build, and the library and test tools voikkospell and voikkohyphenate work correctly.
  • tmispell-voikko was built successfully (Enchant support has not been tested). The resulting tmispell binary works, except for the interactive ncurses frontend.
  • libreoffice-voikko works after some build system hacking.
  • Instructions for building the operating system spell checking service.

The following build procedure assumes that $PREFIX is the directory to which all components will be installed. The instructions are rather brief, but you should be able to follow them if you have experience on building software using GNU autotools. Unreleased versions refer to current SVN trunk.

  • Install Xcode tools from the OS X installation CD.
  • Set some environment variables: 
    export PATH=$PREFIX/bin:$PATH
export PKG_CONFIG_PATH=$PREFIX/lib/pkgconfig
  • Build and install pkg-config 0.23: 
    ./configure --prefix=$PREFIX
make
make install
  • Build and install GNU libiconv 1.9.2: 
    ./configure --prefix=$PREFIX --enable-static
make
make install
  • Build and install GNU gettext 0.17: 
    ./configure --prefix=$PREFIX
make
make install
  • Build and install Glib 2.12.13: 
    CFLAGS="-I$PREFIX/include -L$PREFIX/lib" ./configure --prefix=$PREFIX \
  --enable-static
make
make install
  • Build and install Malaga 7.11: 
    CFLAGS="-I$PREFIX/include -L$PREFIX/lib" ./configure --prefix=$PREFIX
make
make install
  • Build and install Suomi-malaga 1.0: 
    make voikko
make voikko-install DESTDIR=$PREFIX/lib/voikko
  • Build and install libvoikko 2.1: 
    CPPFLAGS="-I$PREFIX/include" CXXFLAGS="-L$PREFIX/lib" ./configure --prefix=$PREFIX \
  --with-dictionary-path=$PREFIX/lib/voikko
make
make install
  • Build and install tmispell-voikko 0.6.3: 
    CPPFLAGS="-I$PREFIX/include" LDFLAGS="-L$PREFIX/lib" \
  ./configure --prefix=$PREFIX --disable-enchant
make
make install
    Copy the provided tmispell.conf.example to $PREFIX/etc/tmispell.conf
  • Build libreoffice-voikko following the instructions in README, and install it by using the LibreOffice extension manager.
